> **Key takeaway:** These transactions require **judgment and additional context** before the correct journal entry can be determined. In each case, the same economic event produces different debit accounts (and sometimes different debit/credit natures) depending on a determining factor that is not evident from the transaction itself. An accounting system processing these transactions must prompt the user for the missing context before posting the entry.
